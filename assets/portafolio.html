Yo, como estudiante

Mi nombre es José Daniel Tejaxún Xicón, este año 2024 estare cursando el tercer año de la carrera de perito en informática en el centro tecnico laboral Kinal, está página es acerca de como es el nivel de preparación con el que se llega a este tercer año.

Proyectos

Tonny's Kinal:

Tonny's Kinal es un programa diseñado para llevar el control de una empresa de eventos para llevar el control de elementos como empleados, lugar de eventos, comidad que se sirve y la logistica en general. Esta desarrolado en el lenguaje de Java SE trabajado en NetBeans, diseñado con SceneBuilder, se almacenan los registros en una base de datos de MySQL y se trabajan reportes con iReport.Más información.

Ahorcado:

Ahorcado es un juego que consiste en adivinar una palabra oculta mientras va apareciendo un personaje por partes, tiene las vidas contadas ganas cuando adivinas la palabara antes de que se te acaben las vidas, esta estructurado con HTML, diseñado con CSS y programado con JavaScript. Más información

Calculadora:

Calculador en programa para realizar operaciones con numeros como sumar, restar, multiplicar, dividir, raiz cuadrada exponentes esta programada en Java SE trabajada en NetBeans y diseñada en SceneBuilder. Más información
